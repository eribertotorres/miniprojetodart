# TaskReport Dart

## Sobre o projeto

O TaskReport Dart é um simulador de tratamento de dados de tarefas em Dart.

O projeto simula o retorno de uma API por meio de uma lista de mapas,
transorma esses dados em objetos e gera relatórios com base no status das tarefas.

## Objetivo

Praticar os principais conceitos do Módulo 01:

- lógica de programção;
- linguagem Dart;
- tipos de dados;
- estruturas de decisão e repetição;
- funções e arrow functions;
- Lists, Maps e Sets;
- métodos de coleção;
- classes, objetos, atributos, métodos e construtores;
- encapsulamento.
- hera~ça e polimorfismo com @override;
- GitHub;
- GitFlow;
- Kanban.

## Como executar

Execute o arquivo 'main.dart' no ambiente utilizado em aula.

No VSCode, procure o void main() e clice em 'Run' para executar o programa
e ter a saída no debug console.
